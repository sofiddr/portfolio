Portfolio

Interactive Maps

Creating interactive maps are part of my coursework in my Masters program. Main tools used are Mapbox JS or Leaflet.

Interactive features were used when more user engagement and bigger pictures are needed, the New York map has a time slider that allows users to see the changes in the taxi demand during the day. The leaflet map which focuses on Santander bike usage before and after COVID-19 comes with a tutorial generated using Bookdown.

Visualization on Tableau

I picked up Tableau back at CityU, this popular reporting tool enabled me to build interactive dashboards with StrawberryNet and later at Urban Renewal Authority.

The preparation work usually start from mining data from the SQL server, followed by cleansing and manipulating tasks in either Python, R or Excel, I personally prefer using Dplyr and stringr in RStudio before importing to Tableau.
